## Chrome如何打開實驗性的功能

在網址上輸入以下內容

```
chrome://flags
```

即可來到實驗性功能的頁面，

例如chrome 90推出了 ``Copy Link To Text``

也可以在這邊啟用。

而其實Copy Link To Text的功能很單純，他就只是改變URL而已

詳細可這篇: [How to highlight text using javascript](https://stackoverflow.com/a/66304412)
